This is a java app for tagging and organizing images and videos. It is made up of a main java file that handles the majority of the functionality, some shell scripts, and python scripts.

install_dependencies.sh runs at startup to ensure that all of the required dependencies are installed and accessible. This is especially important for the python dependencies

video_tagger_CLI.py handles the machine learning processing for facial recognition.
detect_speech.py handles the machine learning processing to create transcripts of video files
